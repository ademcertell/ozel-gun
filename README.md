# Özel Gün Takip

A simple tool to keep track of special occasions and receive reminder notifications.

## Installation

```
git clone
cd ozel-gun
yarn
yarn dev
```

## Features

- [x] Follow special occasions.
- [x] Receive reminder notifications.
- [x] Get gift ideas when the special day arrives.
