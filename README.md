# Özel Gün Takip
Özel günleri takip etmek ve hatırlatma bildirimleri almak için basit bir araç.

## Kurulum
```
git clone
cd ozel-gun
yarn
yarn dev
```

- [x] Özel günlerin takibi.
- [x] Hatırlatma bildirimleri alma.
- [x] Özel günler için hediye fikirleri.
